# Sopes Auto Detailing — Website

A premium, dark-themed website for Sopes Auto Detailing. Built with vanilla HTML, CSS, and JavaScript.

## Design

- **Style:** Dark luxury aesthetic — black, charcoal, deep gray
- **Accent:** Red (#c41e3a)
- **Typography:** Outfit (headings), DM Sans (body)
- **Layout:** Mobile-first, responsive

## Structure

1. **Hero** — Bold headline, CTAs (View Packages, Call for Quote)
2. **About** — Business intro, experience, contact
3. **Packages** — Exterior, Interior, Additional Services (from Payhip)
4. **Why Choose Us** — Expert care, premium results, convenience
5. **Process** — Book → Inspect → Detail → Enjoy
6. **Gallery** — Placeholders for before/after photos
7. **Testimonials** — Review callouts
8. **FAQ** — Common questions
9. **Footer** — Links, contact, payment methods

## Running Locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```
